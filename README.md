# Офисный дэшборд

Простой информационный дэшборд на HTML/CSS/JS с виджетами: погода, курс валют, новости, график и часы.

## Быстрый запуск (локально)

Запустите статический сервер в корне проекта и откройте в браузере:

```bash
cd "c:\Users\dasta\OneDrive\Desktop\Test-67"
python -m http.server 8000
# затем откройте http://localhost:8000
```

Проект — полностью статический, поэтому его можно разместить на GitHub Pages, Netlify или любом другом хостинге статических сайтов.

## Тема (тёмная / светлая)

- Текущая тема сохраняется в `localStorage` под ключом `office-dashboard-theme`.
- Если тема не задана — используется системный `prefers-color-scheme`.
- Кнопка переключения темы в верхней панели меняет тему и обновляет метку.

## Конфигурация API

Откройте `config.js` и при необходимости настройте города, валюты и ключи API. По умолчанию:
- Погода: `wttr.in`
- Валюты: `exchangerate.host`
- Новости: Spaceflight News API (fallback). Для NewsAPI.org вставьте `newsApiKey` в `config.js`.

Пример настройки в `config.js`:

```js
export const config = {
  weatherCity: 'Bishkek',
  currencyBase: 'USD',
  currencySymbols: ['EUR','RUB','KGS'],
  newsApiKey: '', // вставьте, если есть
  refreshIntervalMs: 60000,
};
```
